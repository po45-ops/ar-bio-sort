# AR Bio Sort

เกมจำแนกพืชและสัตว์สำหรับนักเรียน รองรับการลากวางบนคอมพิวเตอร์และอุปกรณ์สัมผัส พร้อมโหมด AR ตรวจจับมือผ่านกล้อง

## เล่นเกม

https://po45-ops.github.io/ar-bio-sort/

## ความสามารถ

- จำแนกสิ่งมีชีวิตเป็นพืชหรือสัตว์
- จับเวลา 60 วินาที พร้อมระบบคะแนน
- รองรับคอมพิวเตอร์ โทรศัพท์ และแท็บเล็ต
- โหมด AR ใช้นิ้วโป้งจีบกับนิ้วชี้เพื่อหยิบ ลาก และปล่อยวัตถุ
- ตำแหน่งวัตถุตรงกับจุดจีบ และชดเชยอัตราส่วนภาพกล้องอัตโนมัติ
- ประมวลผลภาพจากกล้องภายในเบราว์เซอร์

## การใช้งานโหมด AR

เปิดผ่าน Chrome หรือ Edge บน HTTPS แล้วอนุญาตให้เว็บไซต์เข้าถึงกล้อง จากนั้นวางจุดจีบไว้เหนือรูป จีบนิ้วโป้งกับนิ้วชี้เพื่อจับ ลากไปยังกล่อง และแยกนิ้วออกเพื่อปล่อย
