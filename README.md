# AR Bio Sort

เกมจำแนกพืชและสัตว์สำหรับนักเรียน ใช้ภาพถ่ายจริง รองรับการลากวางบนคอมพิวเตอร์และอุปกรณ์สัมผัส พร้อมโหมด AR ตรวจจับมือผ่านกล้อง

## เล่นเกม

https://po45-ops.github.io/ar-bio-sort/

## ความสามารถ

- ภาพถ่ายจริงของพืชและสัตว์ 20 รายการ เก็บไว้ภายในโครงการ
- คุณครูกำหนดเวลาได้ 10–600 วินาที พร้อมปุ่มด่วน 30, 60, 90 และ 120 วินาที
- จับเวลาและคิดคะแนนอัตโนมัติ
- รองรับคอมพิวเตอร์ โทรศัพท์ และแท็บเล็ต
- โหมด AR ใช้ท่ากำมือเพื่อหยิบและลาก แล้วแบมือเพื่อปล่อยวัตถุ
- วงควบคุมแบบกราฟิกไม่มีอิโมจิ และตำแหน่งวัตถุตรงกับกลางฝ่ามือ
- ประมวลผลภาพจากกล้องภายในเบราว์เซอร์

## การใช้งานโหมด AR

เปิดผ่าน Chrome หรือ Edge บน HTTPS แล้วอนุญาตให้เว็บไซต์เข้าถึงกล้อง วางวงควบคุมเหนือรูป กำมือเพื่อจับ ลากไปยังกล่อง และแบมือเพื่อปล่อย
